# Getting Started

Welcome to Haven — a community map of the No Man's Sky universe. This page explains what each tab does and how to upload a submission.

## The Tabs

The top menu is the whole site. Here's what each tab is for.

| Tab | What it is |
|---|---|
| **Dashboard** | The home page. Recent submissions, live stats, and a snapshot of what's happening across the platform. |
| **Systems** | The full searchable catalog. Filter by glyph, galaxy, region, star type, biome, resources — everything. Click any system for the full detail page. |
| **Map** | The 3D galactic map. Spin around, zoom in, click stars to jump to their detail page. |
| **Create** | The submission wizard. Where you add a system you visited. Covered in detail below. |
| **Discoveries** | Fauna, flora, minerals, bases, ships, multitools — anything cool you found *on* a planet. Browse what others have logged or submit your own. |
| **Community Stats** | Per-civilization breakdowns: who's contributed what, how active each community is, top contributors. Click any community to drill in. |
| **Docs** | You're here. Guides for members, civilization leaders, and a deep technical reference. |
| **My Profile** | Visible only when logged in. Your contributions in one place plus default-galaxy / community-tag prefs. |

You don't need an account to browse any of this. An account is only useful if you want a personal profile page.

## Submitting a System

Go to **Create**. The wizard walks you through one screen at a time. Required fields are marked with a red asterisk; everything else is optional and only affects your system's completeness grade (C / B / A / S).

### The required fields

| Field | What goes here |
|---|---|
| **Portal glyphs** | All 12 glyphs of the system. Pick them from the visual hex picker — no typing. |
| **Galaxy** | Which of the 256 NMS galaxies (Euclid, Hilbert, Calypso, etc.). |
| **Reality** | Normal / Permadeath / Custom / etc. |
| **System name** | Whatever the system is called in-game. |
| **Star color** | Yellow, Red, Green, Blue, or Purple. |
| **Economy type** | Trading, Mining, Technology, etc. — or **Abandoned** for empty systems. |
| **Dominant lifeform** | Gek / Vy'keen / Korvax / **None** (no race, never was) / **Abandoned** (empty buildings, race left). |
| **Your Discord username** | So your contribution gets credited. |
| **Your Discord community** | Voyager's Haven, IEA, B.E.S, or "Personal" if you're not in one. |

If you pick **Abandoned**, the wizard skips Economy Tier and Conflict Level for you — abandoned systems don't have those. Otherwise both are required.

### The region-name gate

When you enter your glyphs, the wizard looks up which **region** of the galaxy you're in. If that region doesn't have a community-given name yet, you'll need to **propose one** before you can submit. Every region needs a name; there's no skipping this step. If a name is already there, you'll see it and move on.

### Optional fields

These all bump your completeness grade but never block submission:

- **Spectral classification** (e.g. *G2pf*, *M7*)
- **System description** — a sentence or two on why it's notable
- **Planets and moons** — only the name is required for each; biome, weather, sentinels, flora, fauna, resources, special features, photos are all optional
- **Space station** — uncheck the box if there isn't one
- **Photos** — drop a screenshot per planet if you took one

### What happens next

When you click **Submit**, your submission goes into a review queue scoped to your community. An admin gives it a once-over (usually within a day) and it appears on the live map. If something needs a fix, the admin can tweak it before approving — you won't get rejected over a typo.

## Submitting a Discovery

Discoveries are anything cool you found *on* a planet — fauna, flora, minerals, bases, starships, multitools, or anomalies.

1. Go to **Discoveries**.
2. Click **Submit Discovery** in the top right.
3. Pick the type, link it to a system (or create a stub system on the spot if it isn't in Haven yet), pick the planet/moon or "Space," and fill in the type-specific fields.
4. Submit.

Same review flow as systems — your community admin approves it, then it's live.

## Optional: Make an Account

You can keep submitting forever without an account. Your username gets attached automatically.

If you want one anyway:

1. Submit anything first — this auto-creates a profile under your Discord name.
2. Go to **My Profile** (top-right when logged out, prompt will help) → **Set Password**.
3. Now you can log in. **My Profile** shows everything you've ever submitted, and the wizard auto-fills your default galaxy and community tag.
